# SOLIDWORKS 3D Design Model

## Overview

This repository contains a 3D CAD model designed using **SOLIDWORKS**. The project demonstrates mechanical design principles, parametric modeling, and assembly creation. It can be used for learning, simulation, manufacturing reference, or portfolio purposes.

## Features

* Fully parametric SOLIDWORKS model
* High-quality 3D part and assembly design
* Easy-to-modify dimensions and features
* Engineering drawing included (if available)
* STEP and STL files for compatibility with other CAD software

## Software Used

* SOLIDWORKS 2024 (or your version)
* Windows 10/11

## Repository Contents

* `/Part Files` – Individual SOLIDWORKS part files (.SLDPRT)
* `/Assembly` – Assembly files (.SLDASM)
* `/Drawings` – Engineering drawings (.SLDDRW)
* `/STEP` – STEP files for universal CAD compatibility
* `/STL` – STL files for 3D printing
* `/Images` – Screenshots and rendered images

## Applications

* Mechanical Engineering
* Product Design
* Manufacturing
* 3D Printing
* Educational Projects
